# U+ VOICE — VOC 분류 기준 (Copilot Studio Agent Knowledge)

> 이 문서는 사내 Copilot Studio Agent 'U+VOC voice'의 **지식(Knowledge)** 으로 업로드해 분류·대응의 일관성을 보장하기 위한 기준입니다.
> (앱 소스 `src/classify.js`에서 자동 생성 — 기준이 바뀌면 재생성하세요.)

## 1. VOC구분 4그룹
1. **장애/오류**
2. **성능**
3. **개선 요청/희망**
4. **단순 문의/불만/기타**

### 정형 그룹의 닫힌 분류(구분2)
- **장애/오류**: 로그인불가/로그인풀림 · 앱/웹 기능오류 · 앱/웹 접속불가 · 앱/웹화면 데이터 정합성 이슈 · 기타
- **성능**: 앱/웹 속도 느림 · 앱/웹 백화 현상
- **개선 요청/희망**: 앱/웹 기능 개선 · 회원/로그인 개선 · 기타
> 정형 3그룹(장애/오류·성능·개선 요청/희망)은 위 닫힌 분류로 매핑합니다. 그 외는 '단순 문의/불만/기타'(열림 그룹)로 보고 아래 22표준분류로 분류합니다.

## 3. 대응영역 트리 (depth1 › depth2) — depth2도 아래 값에서 선택
- **MY**: 가입정보관리 · 요금/납부/청구 · 회원/로그인/ID · 휴대폰 결제 · 데이터 · 고객지원 · 자녀 통신요금 관리 · 메뉴/GNB/위젯 · 결합할인
- **검색/챗봇**: 검색/챗봇
- **혜택/멤버십**: VIP콕(영화/구독/제휴) · 바코드/메뉴 · 유플미션/출석체크 · 멤버십(등급/정책) · 유플투쁠
- **상품/스토어**: 기타(유독) · 로밍 · 모바일 가입 · 유심/이심(eSIM)/너겟 · 모바일 요금제 · 모바일 부가서비스 · 홈 요금제/홈 부가서비스 · 액세서리/라이브

### 영역별 담당(참고)
- MY → 김민수 · MY서비스팀
- 검색/챗봇 → 이서연 · AI검색팀
- 혜택/멤버십 → 박지훈 · 멤버십팀
- 상품/스토어 → 최유진 · 커머스팀

## 5. 분류별 권장 응대·조치 (응대 톤·내부 액션 기준)
### 네트워크/통신품질/와이파이
- 고객 응대: 통신 품질 불편을 확인했습니다. 사용 지역·시간대를 확인해 망 품질을 점검하고, 필요 시 중계기 점검을 요청드리겠습니다.
- 내부 조치: 발생 지역·시간 확인 → 망 품질 점검 의뢰 후 회신

### 인터넷·통신속도 불만
- 고객 응대: 인터넷·데이터 속도 불편을 확인했습니다. 회선 상태와 속도를 점검해 원인을 안내드리겠습니다.
- 내부 조치: 회선 속도 점검 → 원인 안내, 필요 시 기사 방문

### 앱·웹 이용문의
- 고객 응대: 앱·웹 이용 방법 문의를 확인했습니다. 해당 메뉴 위치와 이용 방법을 안내드리겠습니다.
- 내부 조치: 이용 방법 안내

### 요금제
- 고객 응대: 요금제 관련 문의를 확인했습니다. 사용 패턴에 맞는 요금제를 비교해 안내드리겠습니다.
- 내부 조치: 사용량 기반 요금제 진단·추천

### 해지/약정/위약금
- 고객 응대: 해지·약정 관련 문의를 확인했습니다. 약정 잔여·예상 위약금을 정확히 확인해 안내드리겠습니다.
- 내부 조치: 약정/위약금 조회 → 정확 금액·절차 안내

### 가입/개통/결합
- 고객 응대: 가입·개통·결합 관련 문의를 확인했습니다. 진행 조건과 절차를 안내드리겠습니다.
- 내부 조치: 가입/결합 조건 확인 후 절차 안내

### 부가서비스
- 고객 응대: 부가서비스 관련 문의를 확인했습니다. 가입·해지·이용 방법을 안내드리겠습니다.
- 내부 조치: 부가서비스 가입/해지 처리·안내

### 데이터(사용량/선물/충전)
- 고객 응대: 데이터 관련 문의를 확인했습니다. 사용량·충전·선물 방법을 안내드리겠습니다.
- 내부 조치: 데이터 사용량 조회·충전/선물 안내

### 로밍
- 고객 응대: 로밍 관련 문의를 확인했습니다. 출국 전 로밍 요금제와 해외 이용 방법을 안내드리겠습니다.
- 내부 조치: 로밍 요금제/이용 방법 안내

### 유심/이심/IMSI
- 고객 응대: 유심·이심 관련 문의를 확인했습니다. 교체·전환 절차와 준비물을 안내드리겠습니다.
- 내부 조치: 유심/이심 교체·전환 절차 안내

### 단말/기기/액세서리
- 고객 응대: 단말기 관련 문의를 확인했습니다. 기기 설정·변경 방법을 안내드리겠습니다.
- 내부 조치: 기기 설정/변경 안내

### 멤버십/쿠폰/혜택/VIP콕
- 고객 응대: 멤버십·혜택 관련 문의를 확인했습니다. 쿠폰·혜택 사용 방법과 적용 여부를 확인해 안내드리겠습니다.
- 내부 조치: 쿠폰/혜택 적용 확인·안내

### 설치/AS(홈상품)
- 고객 응대: 설치·AS 관련 문의를 확인했습니다. 일정 확인 후 기사 방문을 안내드리겠습니다.
- 내부 조치: 설치/AS 일정 확인·배정

### IPTV/셋톱박스
- 고객 응대: IPTV·셋톱박스 관련 문의를 확인했습니다. 증상 점검 후 해결 방법을 안내드리겠습니다.
- 내부 조치: 셋톱박스 점검·조치 안내

### 상담/고객지원
- 고객 응대: 상담 연결 불편을 확인했습니다. 빠르게 담당자가 연결되도록 조치하겠습니다.
- 내부 조치: 상담 이력 확인 → 우선 연결/콜백

### 매장/대리점
- 고객 응대: 매장 이용 관련 불편을 확인했습니다. 사실관계를 확인해 안내드리겠습니다.
- 내부 조치: 매장 사실관계 확인 후 회신

### 회원/로그인/인증
- 고객 응대: 로그인·인증 문제를 확인했습니다. 본인확인 후 정상 이용되도록 도와드리겠습니다.
- 내부 조치: 계정/인증 상태 점검 → 복구 안내

### 요금/청구/납부/환불
- 고객 응대: 요금·청구·납부 관련 문의를 확인했습니다. 청구 내역을 확인해 정확한 금액과 납부 방법을 안내드리겠습니다.
- 내부 조치: 청구/납부 내역 확인 → 정정·환불 필요 시 처리

### 휴대폰결제/소액결제
- 고객 응대: 휴대폰결제 관련 문의를 확인했습니다. 한도·이용 내역을 확인해 안내드리겠습니다.
- 내부 조치: 결제 한도/내역 확인·안내

### 유독/모바일TV/익시오/스마트홈
- 고객 응대: 유독·모바일TV 등 서비스 문의를 확인했습니다. 이용·해지 방법을 안내드리겠습니다.
- 내부 조치: 구독 서비스 이용/해지 안내

### 배송
- 고객 응대: 배송 관련 문의를 확인했습니다. 배송 현황을 확인해 안내드리겠습니다.
- 내부 조치: 배송 현황 확인·안내

### 검색/챗봇/AI
- 고객 응대: 검색·챗봇 이용 관련 문의를 확인했습니다. 정확한 결과가 나오도록 점검하겠습니다.
- 내부 조치: 검색/챗봇 동작 점검

## 6. 심각도(severity) 판정 기준
- **높음(High)**: 장애/오류 그룹 전체, 또는 '이중·중복청구·위약금·환불·미납·접속불가·먹통·불가·끊김·안터' 등 고위험 신호어
- **보통(Medium)**: 성능 저하, 또는 '불만·짜증·항의·최악·화남·불편' 등 불만 표현
- **낮음(Low)**: 일반 문의 수준

## 7. 출력 형식 (사이트 연동용 구조화 JSON)
"JSON으로/구조화 결과/사이트용으로" 요청 시, 설명·머리말·코드펜스 없이 아래 객체 하나(또는 여러 건이면 `{ "items": [ … ] }` 배열)만 출력합니다.

  "classification": {

    "category": "위 22표준분류 중 하나(철자 그대로)",
    "depth1": "MY | 검색/챗봇 | 혜택/멤버십 | 상품/스토어",
    "depth2": "대응영역 트리의 depth2 값",
    "severity": "높음 | 보통 | 낮음",
    "sentiment": "긍정 | 중립 | 부정",
    "urgency": "높음 | 보통 | 낮음",
    "signals": ["판단 근거 신호어"],
    "relatedMenu": "관련 메뉴/경로 (예: 마이페이지_가입정보)",
    "errorType": "유큐브/그외 연동오류 | 앱/웹 기능오류 | 데이터 정합성 | 접속불가 | 성능 지연 | 기획/정책 | 기타"
  },

  "labels": ["VOC", "전사IT업무요청"],

### 규칙
- 표/트리에 없으면 가장 가까운 항목 + 값 뒤에 `(확인 필요)`.
- 정책·수치(위약금·요율·일정)는 근거 없으면 `(확인 필요)`.
